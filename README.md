# A2A Essay Pipeline

FastAPI-сервис с A2A (Agent-to-Agent) конвейером для написания и редактирования эссе с использованием OpenAI GPT-4.

## Возможности

- **Агент-писатель эссе**: Генерирует эссе на заданные темы
- **Агент-редактор стиля**: Улучшает стиль и ясность текста
- **A2A интеграция**: Агенты общаются через протокол Google Agent-to-Agent
- **OpenAI интеграция**: Использует GPT-4 для генерации и редактирования контента

## Предварительные требования

- Python 3.10+
- Ключ API OpenAI
- Менеджер пакетов uv

## Установка

```bash
# Установка uv, если не установлен
pip install uv

# Синхронизация зависимостей
uv sync
```

## Настройка

Установите ваш ключ API OpenAI:

```bash
export OPENAI_API_KEY="your-api-key-here"
```

## Запуск сервера

```bash
# Запуск с uvicorn
uv run uvicorn app.server:app --host 0.0.0.0 --port 8000

# Или через Python модуль
uv run python -m app.server
```

## Запуск клиента

```bash
uv run python client/run_essay_workflow.py
```

Клиент запросит тему эссе и запустит A2A рабочий процесс.

## Архитектура

1. Клиент отправляет запрос агенту-писателю эссе через A2A
2. Агент-писатель генерирует первоначальный контент эссе с помощью OpenAI
3. Агент-писатель вызывает агента-редактора стиля через A2A для улучшения текста
4. Агент-редактор дорабатывает стиль эссе с помощью OpenAI
5. Финальный результат возвращается клиенту
